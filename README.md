# Portfolio

Product designer portfolio built with React, TypeScript, and Vite.

 **Live site:** [portfolio-apoly.vercel.app](https://portfolio-apoly.vercel.app/)

 **Storybook:** [Design System Docs](https://a123poly-dev.github.io/portfolio/?path=/docs/common-heading--docs)

## Prerequisites

- [Node.js](https://nodejs.org/) v20+
- npm

## Getting Started

```bash
git clone https://github.com/a123poly-dev/portfolio.git
cd portfolio/app
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

## Storybook

```bash
cd app
npm run storybook
```

Opens at `http://localhost:6006`.

## Tech Stack

React 19 路 TypeScript 路 Vite 6 路 SCSS Modules 路 React Router 7 路 Storybook 10
